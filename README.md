# byr-consulting-app

Aplicación Flutter para BYR Consulting, diseñada para gestionar registros de empleados y reportes semanales, mensuales y bimestrales.

## 🛠️ Requisitos

- Flutter ≥ 3.x  
- Dart ≥ 2.x  
- Android SDK (minSdk 23)  
- Xcode (para iOS)  
- Una cuenta de Firebase configurada con Firestore
